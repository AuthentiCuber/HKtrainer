# HKtrainer
A hotbar hotkey trainer for 1.16.1. Created to make it easier/faster to learn new keybinds e.g. for MCSR.

## Installation
Download the [latest release](https://github.com/AuthentiCuber/HKtrainer/releases/latest) and put the zip file in the `datapacks` folder of a new creative world.

## Usage
You will be given a snowball in a random hotbar slot. Hotkey to it and throw it. Repeat.
